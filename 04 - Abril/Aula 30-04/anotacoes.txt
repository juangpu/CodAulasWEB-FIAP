Posicionar containers(elementos)
tag html recebe conteúdo == caixa
Flexbox == faz o posicionamento baseado no elemento PAI == é o container onde os outros elementos estão
O elemento pai receberá a propriedade display com valor FLEX == por padrão os filhos ficarão um ao lado do outro 
ocupando o espaço do pai == NUNCA POSSO ESQUECER = FLEX É APLICADO AO PAIIIIIIIII.
quando elemento pai recebe a propriedade == Flex container 
os filhos desse container são chamados de flex-items
por padrão propriedade flex-direction: row - column (um abaixo do outro)
alinhamento pela horizontal / vertical
vertical = align-items = flex-start(padrão) 
horizontal = justify-content = flex-start
space-between ==> alinha os filhos pelas extremidades do pai
space-around ==> alinhar os filhos pelo centro do pai
